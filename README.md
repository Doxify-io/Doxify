# Doxify.io — Free Professional Document Generator

🌍 Live at: https://doxify.io

## What is Doxify?

Doxify is a completely free professional document 
generator for workers, freelancers and HR teams worldwide.

No login. No watermark. No subscription. Forever free.

## Documents You Can Generate

| Document | Use Case |
|---|---|
| 💼 Salary Slip | Monthly payroll document |
| 📜 Salary Certificate | For visa and bank applications |
| ✅ NOC Letter | No Objection Certificate |
| 🏅 Experience Letter | For departing employees |
| ✈️ Visa Cover Letter | For any visa application |
| 🤝 Freelance Contract | Client agreements |
| 🏦 Bank Statement Letter | Proof of income |

## Features

- ✅ Add company logo and letterhead
- ✅ Add digital signature
- ✅ Preview before downloading
- ✅ Clean professional PDF
- ✅ Works on mobile
- ✅ No login required
- ✅ No watermark
- ✅ 100% free forever
